Aimlessly plodding off in my own direction. The plan is to 
get Cyberspace running on a Flipper Zero, because it has no 
point and I love it. This is the Python prototype built to 
figure out the API first. It works. Next step is the Flipper app.

cyberspace.online -- you'll need an account and API access.
C-Mail @genghis_khan to request it.


Requirements

    Python 3.8 or later
    A Cyberspace account with API access


Installation

    git clone https://github.com/Mr-Mammoth/CyberFlipperPirate.git
    cd CyberFlipperPirate
    python3 -m venv cyberspace-env
    source cyberspace-env/bin/activate
    pip install requests

Works on macOS, Linux, and Windows (WSL).


Usage

    source cyberspace-env/bin/activate
    python3 cyberspace_flipper.py

You'll be prompted for your password. It is not stored anywhere.
